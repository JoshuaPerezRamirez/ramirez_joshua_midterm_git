## Midterm Examination Reflection

### 1. What is Git used for?
Git is used to track changes in files and manage different versions of a project.
It also helps developers work on projects efficiently without losing their work.

### 2. What is the difference between Git and GitHub?
Git is a version control tool that works on your local computer.
GitHub is an online platform where Git repositories are stored, shared, and collaborated on.

### 3. Why is branching useful?
Branching allows you to work on new features without affecting the main project.
It makes testing and development safer before merging changes.

### 4. What command did you use to upload your local commits to GitHub?
I used the git push -u origin main command to upload my local commits to GitHub.
After the first push, I used git push origin main to upload new changes.

### 5. What did you learn from this activity?
I learned how to use Git commands to manage project versions and track changes.
I also learned how to use GitHub to store projects online and collaborate more effectively.
